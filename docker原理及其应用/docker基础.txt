docker 利用linux内核的几个特性来实现功能
利用linux的命名空间（namespaces） 为docker容器提供操作系统层面的隔离
    liunx控制组（control groups）为docker容器提供硬件层面的隔离
    liunx联合文件系统（union file system）利用分层思想管理镜像和容器
    意味着 docker 只能在linux上运行

docker 版本{
    docker-ce 社区版
    docker-ee 企业版
}

docker image inspect afd6
 查看镜像详细信息

docker 中 镜像，容器，网络等等都是对象，可使用docker命令进行增删改查
 docker 网络管理
